8. Indica la diferencia entre los siguientes métodos, y demuestra su uso con algunos arrays:
  • Array.prototype.forEach()
    • El método forEach() ejecuta una función por cada uno de los elementos del array.
  • Array.prototype.every()
    • El método every() verifica si TODOS los elementos del array cumplen con la condición introducida. Devuelve true o false.
  • Array.prototype.some()
    • El método some() verifica si ALGÚN elemento del array cumple con la condición introducida. Devuelve true o false.
  • Array.prototype.filter()
    • El método filter() devuelve un array con todos los elementos que cumplan con la condición asignada.
